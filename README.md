# ReLoader 🚀

![Version](https://img.shields.io/badge/version-2.6.1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)

**ReLoader** — это мощная и легкая утилита для автоматизации запуска приложений и управления процессами в Windows. Идеально подходит для системных администраторов, геймеров и продвинутых пользователей.

## ✨ Основные возможности

- 🚀 **Умный запуск**: Автоматический запуск групп приложений при старте или по нажатию кнопки.
- 🌐 **Cloud Config**: Поддержка внешних конфигураций через URL (GitHub, Gist и т.д.) или сетевые пути.
- 🎯 **Global Hotkeys**: Запускайте свои программы из любого окна с помощью `Alt + R` (настраиваемо).
- 🛡️ **Watchdog (Сторож)**: Автоматически перезапускает упавшие или закрытые приложения.
- 🧠 **Smart Launch**: Предотвращает запуск дубликатов уже открытых программ.
- 🖱️ **Drag-and-Drop**: Добавляйте новые программы в список запуска, просто перетащив их на окно ReLoader.
- 👻 **Stealth Mode**: Работайте полностью в фоновом режиме (только трей и горячие клавиши).
- 🎨 **Кастомизация**: Настройка цветов интерфейса, центрирование окна и звуки запуска.
- 📊 **Progress Ring**: Наглядная визуализация процесса запуска приложений.

## 📥 Установка

1. Перейдите в раздел [Releases](https://github.com/komoliddin/ReLoader/releases).
2. Скачайте последнюю версию `ReLoader.exe`.
3. Запустите программу. Файл конфигурации `ReLoaderCFG.txt` будет создан автоматически при первом запуске.

## ⚙️ Настройка (ReLoaderCFG.txt)

Программа настраивается через текстовый файл. Основные параметры:
- `CONFIG=url/path`: Путь к внешнему файлу настроек (HTTP/UNC/Local).
- `LEFT=CENTER`: Центрирование окна по горизонтали.
- `DELAY`: Задержка между запусками (в секундах).
- `HOTKEY`: Ваша комбинация клавиш (например, `Ctrl+Shift+L`).
- `WATCHDOG=1`: Включить режим контроля за процессами.
- `SMART_LAUNCH=1`: Включить защиту от дубликатов.
- `COLOR_TEXT`: Цвет текста (например, `White` или `#FF5733`).
