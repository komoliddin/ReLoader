# ReLoader 🚀 — Ультимативный менеджер запуска для Windows

![Version](https://img.shields.io/badge/version-2.6.2.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)

**ReLoader** — это компактная, но мощная утилита на C# для автоматизации рабочего окружения. Она позволяет управлять группами приложений, следить за их состоянием и синхронизировать настройки через облако.

---

## ✨ Основные возможности

- 🚀 **Умный запуск**: Автоматический запуск групп приложений при старте или по нажатию кнопки.
- 🌐 **Cloud Config**: Поддержка внешних конфигураций через URL (GitHub, Gist и т.д.) или сетевые пути.
- 🎯 **Global Hotkeys**: Запускайте свои программы из любого окна с помощью `Alt + R` (настраиваемо).
- 🛡️ **Watchdog (Сторож)**: Автоматически перезапускает упавшие или закрытые приложения.
- 🧠 **Smart Launch**: Предотвращает запуск дубликатов уже открытых программ.
- 🖱️ **Drag-and-Drop**: Добавляйте новые программы в список запуска, просто перетащив их на окно ReLoader.
- 👻 **Stealth Mode**: Работайте полностью в фоновом режиме (только трей и горячие клавиши).
- 🎨 **Кастомизация**: Настройка цветов интерфейса, центрирование окна и звуки запуска.
- 📊 **Progress Ring**: Наглядная визуализация процесса запуска приложений.

## 📥 Установка

1. Скачайте `ReLoader.exe` из раздела [Releases](https://github.com/komoliddin/ReLoader/releases).
2. Поместите файл в любую папку (рекомендуется создать отдельную папку, так как программа создаст файл настроек и логов).
3. Запустите программу. При первом старте она автоматически создаст эталонный `ReLoaderCFG.txt` с подробными комментариями.

---

## 🖱 Интерфейс и управление

- **Круг в центре**: Основная кнопка. Клик запускает последовательность программ.
- **Drag-and-Drop**: Просто перетащите любой `.exe` файл на окно программы, и он будет добавлен в список запуска `START`.
- **Контекстное меню (ПКМ по окну или иконке в трее)**:
  - *Активные процессы*: Список программ из вашего конфига, которые запущены в данный момент.
  - *Стелс-режим*: Скрывает основное окно, оставляя работу только через горячие клавиши и трей.
  - *Закрепить окно*: Запрещает случайное перемещение окна мышкой.
  - *Проверить обновления*: Ручная проверка новой версии на сервере.

## ⚙️ Настройка (ReLoaderCFG.txt)

Файл конфигурации поддерживает следующие ключевые параметры:

| Ключ | Значение | Описание |
| :--- | :--- | :--- |
| `DELAY` | `1, 2, 3...` | Задержка в секундах между запуском приложений. |
| `HOTKEY` | `Alt+R` | Глобальная комбинация клавиш для запуска. |
| `OPACITY` | `25-100` | Прозрачность окна программы. |
| `LEFT/TOP` | `100 / CENTER` | Координаты окна. `CENTER` — автоматическое центрирование. |
| `WATCHDOG` | `1 / 0` | "Сторож": если программа из списка закрыта, ReLoader запустит её снова. |
| `SMART_LAUNCH`| `1 / 0` | Не запускает приложение, если его процесс уже виден в системе. |

### Списки приложений:
- `START=путь`: Запуск только при клике по кнопке или горячей клавише.
- `AUTOSTART=путь`: Запуск автоматически при старте ReLoader.
- `STARTBOTH=путь`: Запуск и при старте, и при клике.
- `TASKKILL=имя.exe`: Завершение процесса перед началом новой последовательности запуска.

## 🌐 Облачная синхронизация (Cloud Config)

ReLoader поддерживает загрузку настроек из внешних источников. Это идеально для синхронизации настроек между несколькими ПК или для системных администраторов.

Добавьте в локальный `ReLoaderCFG.txt` строку:
`CONFIG=https://your-server.com/reloader_base.txt`

**Особенности:**
- Поддерживаются ссылки `http/https`, сетевые пути `\\server\share\` и локальные пути.
- При наличии `CONFIG=` программа скачивает внешние настройки, но **сохраняет** ваши локальные списки программ (`START`, `AUTOSTART` и т.д.).
- Локальные и облачные списки объединяются (без дубликатов).

## 🧹 Удаленное управление (Blacklist)

Если вы администратор и хотите удаленно запретить запуск определенных программ у пользователей, используйте префикс `REMOVE_` в **облачном** файле:

`REMOVE_START=C:\Games\Doom.exe`
`REMOVE_TASKKILL=chrome.exe`

Программа автоматически удалит указанные пути из локальных списков пользователя при следующем обновлении конфигурации.
