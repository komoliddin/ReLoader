# ReLoader - Журнал изменений (Change Log)

Все важные изменения в проекте фиксируются здесь.

## [2.6.2.0] - 2026-05-02
### Добавлено
- **Умное слияние (Smart Merge)**: Теперь при использовании `CONFIG=` ваши локальные списки программ (`START`, `AUTOSTART` и др.) не удаляются, а объединяются с облачными.
- **Удаленное управление (Blacklist)**: Поддержка префикса `REMOVE_` в облачных конфигах для принудительной очистки локальных списков пользователя (административная функция).
- **Поддержка STARTBOTH**: Локальное сохранение списка «Оба» при синхронизации.

### Изменено
- Ключ `PROFILE=` переименован в `CONFIG=` для соответствия техническим стандартам.
- `README.md` полностью переписан и теперь является полноценной инструкцией по эксплуатации.

## [2.6.1.0] - 2026-05-02
### Добавлено
- **Поддержка внешних конфигураций**: Новый параметр `CONFIG=` позволяет загружать настройки из URL (HTTP/HTTPS), сетевых путей (UNC) или локальных файлов.
- **Центрирование окна**: Параметры `LEFT=CENTER` и `TOP=CENTER` для автоматического позиционирования окна по центру экрана.
- **Разделители в меню**: Улучшена визуальная структура контекстного меню.

### Исправлено
- Исправлено отображение галочки для пункта «Создать ярлык на рабочем столе».
- Оптимизирован порядок пунктов в контекстном меню для более удобного доступа.
- Переработан механизм создания конфигурации по умолчанию (теперь со всеми подсказками).

## [2.6.0.1] - 2026-05-01
### Изменено
- Из контекстного меню удалены пункты "Редактировать список приложений" и "Полная очистка".

## [2.6.0.0] - 2026-02-01
### 🚀 Глобальное обновление
Это крупнейшее обновление в истории проекта, включающее полную переработку архитектуры и множество новых функций.

### Добавлено
- **Модульная система**: Код разделен на независимые блоки (`AppLauncher`, `ConfigManager`, `Logger`) для повышения надежности.
- **Горячие клавиши**: Добавлена поддержка глобальных сочетаний клавиш (по умолчанию `Alt+R`).
- **Стелс-режим**: Возможность работы без окна программы (только в трее).
- **Индикатор прогресса**: Добавлено графическое кольцо (Progress Ring), отображающее ход запуска приложений.
- **Drag-and-Drop**: Добавление программ в список перетаскиванием файлов на форму.
- **Watchdog**: Режим автоматического контроля и перезапуска процессов.
- **Smart Launch**: Защита от запуска нескольких копий одной и той же программы.
- **Динамическое меню**: Теперь в трее отображается список реально запущенных приложений.
- **Безопасность**: Автоматическое создание бэкапа настроек (`ReLoaderCFG.txt.bak`).
- **Переезд на GitHub**: Обновления теперь скачиваются напрямую с GitHub.

### Изменено
- Весь процесс запуска теперь полностью асинхронный (UI не зависает).
- Расширена система логирования всех действий пользователя.
- Улучшена совместимость с Windows 10/11 (автоматическая настройка прав администратора).

### Исправлено
- Исправлены критические ошибки при работе с реестром.
- Исправлена логика удаления и создания ярлыков.
